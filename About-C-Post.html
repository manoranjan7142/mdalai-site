C Variables

Published 3 months ago5 min read4 comments

Variable is a name of memory location which can store some data. Sometime we called as Container . if we give some name into that(container) which contain some value then we called as variables .

Syntax Code Example

                    
                        #include < stdio.h>
                            int main(){
                                int number = 25;
                                char star = " * " ;
                                int age = 22;
                                folat pi = 3.14 ;
                            }

Rules :-

  • Variable name case sensitive like (a , A)
  • First character is alphabet or " _ "
  • No comma / blank space
  • No other symbols
